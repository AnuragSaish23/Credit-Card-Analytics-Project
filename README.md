
# Credit Card Spend & Customer Behavior Analysis

## Overview
This project is an end-to-end data analytics case study focused on understanding credit card customer spending and transaction behavior. The objective is to simulate real-world analytics workflows used in financial services companies by transforming raw transaction data into actionable business insights.

## Business Problem
Credit card companies need to understand how customers use their cards, where they spend, and how spending behavior varies across customer segments. Such insights help in improving customer engagement, managing risk, and driving data-informed business decisions.

## Project Objectives

- Analyze customer spending and transaction patterns
- Segment customers based on usage behavior
- Identify trends and insights that support business decision-making
- Build dashboards to clearly communicate insights to stakeholders

## Tech Stack

- **Database:** MySQL  
- **Querying & Analysis:** SQL  
- **Data Analysis:** Python (Pandas, NumPy, Matplotlib, Seaborn)  
- **Visualization:** Power BI  
- **Version Control:** Git & GitHub
